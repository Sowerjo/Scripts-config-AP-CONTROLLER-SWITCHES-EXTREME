CONFIGURAÇÃO INICIAL VIA CLI

CRIANDO REDE WIRELESS

self
wlan (nome do perfil de rede)
description (descrição da rede)
ssid (nome de propagação)
vlan 1
bridging-mode local
encryption-type ccmp
authentication-type none
wpa-wpa2 psk 0 (sua senha de 8 digitos ou mais)
use wlan-qos-policy (nao e obrigatorio)
commit
commit write
write memory
